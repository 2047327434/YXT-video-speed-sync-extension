# 📚 云学堂自动挂课助手 (Chrome Extension)

一个专为 **云学堂（yunxuetang）** 设计的 Chrome 浏览器插件，支持**自动播放视频、视频结束后自动切换下一课、后台静默挂机、防检测机制**。

---

## ✨ 功能特性

| 功能 | 说明 |
|------|------|
| ▶️ **自动播放** | 页面加载后自动检测视频并播放，无需手动点击 |
| ⏭️ **自动切课** | 视频播放完毕后自动点击"下一个"，无缝切换下一节课 |
| 🔇 **后台静默** | 自动静音，视频在后台播放，不影响你做其他事情 |
| 🛡️ **失焦保活** | 页面切换到后台/标签页失焦时，视频不会被暂停 |
| 🎭 **反检测模拟** | 随机小暂停（1~3秒），模拟真实用户观看行为，绕过反作弊检测 |
| ❌ **弹窗自动关闭** | 自动关闭提示弹窗（不关闭考试、练习等关键弹窗） |
| 📊 **实时统计** | 显示运行时长、已完成课程数、当前视频进度 |

## 🚀 安装方法

1. 打开 Chrome，地址栏输入 `chrome://extensions/`
2. 右上角开启「**开发者模式**」
3. 点击「**加载已解压的扩展程序**」
4. 选择 `video-speed-sync-extension` 文件夹

---

## 📖 使用教程

### 开始挂机

1. 打开云学堂课程页面（确保在视频播放页）
2. 点击浏览器工具栏的 📚 图标打开控制面板
3. 点击「**▶️ 开始挂机**」
4. 插件会自动：播放视频 → 视频结束 → 点击"下一个" → 播放下一课 → 循环

### 后台模式

- 勾选「**🔇 后台静默模式**」后，视频会自动静音
- 你可以切换到其他标签页或最小化浏览器，视频会继续播放
- 页面失焦时插件会自动保活，防止网站暂停视频

### 快捷操作

| 按钮 | 功能 |
|------|------|
| ⏭️ 跳过当前课 | 立即点击"下一个"，跳过当前正在播放的视频 |
| 🔄 刷新状态 | 手动刷新面板上的视频进度和状态 |

## 🛠️ 工作原理

```
页面加载
    │
    ▼
检测视频元素 ──否──► 等待2秒后重试
    │
    是
    ▼
自动播放视频（处理浏览器自动播放限制）
    │
    ▼
监听 video.ended 事件 ──或── 监听 timeupdate（currentTime >= duration - 3s）
    │
    ▼
等待 2~3 秒缓冲
    │
    ▼
点击"下一个"按钮
    │
    ▼
回到"检测视频元素"步骤，循环执行
```

### 防检测机制

| 机制 | 说明 |
|------|------|
| **失焦保活** | 覆盖 `visibilitychange` 事件，页面切换到后台时自动恢复播放 |
| **随机暂停** | 每 30~60 秒有 5% 概率暂停 1~3 秒，模拟真实用户偶尔离开 |
| **弹窗过滤** | 自动关闭提示类弹窗，但保留考试/练习/测评等关键弹窗 |

## ⚠️ 注意事项

1. **请勿关闭标签页**：挂机期间请保持云学堂标签页打开，可以切换到其他标签页或最小化浏览器
2. **考试弹窗**：插件会自动识别并保留考试、练习、测评等关键弹窗，不会误关闭
3. **最后一课**：当找不到"下一个"按钮时，插件会自动停止并提示"所有课程已完成"
4. **网络问题**：如果视频加载失败或卡顿，插件会每 2 秒检测一次并自动重试播放

## 🔧 常见问题

**Q: 视频没有自动播放？**
> A: 部分浏览器会阻止自动播放。插件会尝试点击播放按钮，如果仍无法播放，请手动点击一次播放按钮，之后插件会接管。

**Q: 为什么视频播完后没有自动切课？**
> A: 检查"下一个"按钮是否在页面上可见。有些课程页面结构不同，插件可能无法定位按钮。可以手动点击"⏭️ 跳过当前课"。

**Q: 后台播放会被检测吗？**
> A: 插件内置了反检测机制，包括失焦保活和随机暂停模拟。但无法 100% 保证绕过所有检测，建议配合正常使用。
